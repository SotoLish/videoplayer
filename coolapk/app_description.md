# VideoPlayer - 酷安发布文案

## 应用名称
VideoPlayer（视频播放器）

## 一句话简介
极简纯粹的本地视频播放器，专注播放体验。

## 应用简介（详细版）

VideoPlayer 是一款纯粹的本地视频播放器，没有广告、不会联网、不收集任何数据。

**核心特性**

- 沉浸式全屏播放，控制栏 3 秒自动隐藏
- 滑动快进快退，水平拖拽即可跳跃，实时预览跳跃时间
- 六档倍速播放：0.5x 到 2.0x，追剧看教程随心切换
- 拖拽进度条精准定位，支持任意跳转
- 适配 Android 7.0+，深色主题护眼观感
- 赛博朋克风图标，简约不简陋

**适用场景**

- 播放手机本地下载的视频文件
- 倍速刷教程、网课
- 追剧看电影，沉浸不被打扰

**技术栈**

Kotlin + Jetpack Compose + Media3 ExoPlayer，纯原生 Android 开发。

**隐私说明**

本应用完全离线运行，不请求网络权限，不收集任何个人信息，不需要注册账号。
